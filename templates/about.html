About

Dot Gov Domains is a project intended to provide an easily accessible means of sifting through state and federal .gov domains, and examining the assorted properties involved with different domains. Currently, you can examine either federal or state domains. The state domain set excludes domains from U.S. territories, the District of Columbia and tribal nations. Several data sets including these domains can be found in the project's Github repo. Data comes from the GSA's .Gov Domains API and 18F's HTTPS pulse project. The data has been lightly edited (mostly for presentational purposes), but there may still be inaccuracies.

This is an open-source project. If you notice any issues or have suggestions for improvements, feel free to contribute via Github or send an email. You can also fork the project, steal the idea for another country's version of '.gov,' or use any of the aforementioned associated data as you wish.
